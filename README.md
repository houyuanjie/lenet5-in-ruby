# LeNet-5 in Ruby

## Requirements

- Ruby 3 [version](.ruby-version)
- Python 3.12

## Setup

Install Ruby dependencies:

```powershell
bundle install
```

## Run

Run ruby main.rb script:

```powershell
ruby main.rb
```

## Native Ext.

Run rake-compiler:

```powershell
# ridk enable
rake compile
```
